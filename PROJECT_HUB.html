Tuần Nội dung Deliverable Trạng thái
1 Setup môi trường, cài dependencies, test kết nối RTSP với camera Script test_rtsp.py chạy được ⏳ Đang làm
2 Xây module nhận diện khuôn mặt (face_recognizer.py) Nhận diện ≥ 80% Chưa bắt đầu
3 Xây module phát hiện biển số xe — YOLOv8 + EasyOCR OCR biển số VN ≥ 70% Chưa bắt đầu
4 Camera Manager — kết nối đa hãng, RTSP templates, auto-reconnect API cameras hoạt động Chưa bắt đầu
5 Tích hợp AI vào FastAPI qua WebSocket — real-time stream WebSocket stream chạy Chưa bắt đầu
6 Frontend dashboard — live view, bounding box overlay, quản lý camera Giao diện xem camera Chưa bắt đầu
7 Alert email, whitelist, báo cáo thống kê, tối ưu hiệu năng Email cảnh báo hoạt động Chưa bắt đầu
8 Testing, Docker, deploy, quay video demo, hoàn thiện README Portfolio hoàn chỉnh Chưa bắt đầu

🔧 Công nghệ sử dụng

FastAPI (backend)
JWT + Stripe (auth & payment)
SQLite (database)
OpenCV (RTSP stream)
face-recognition (khuôn mặt)
YOLOv8 + EasyOCR (biển số)
WebSocket (real-time)
Docker (deployment)
ℹ️ Ghi chú cho Mentor: Dự án này kế thừa nền tảng Chart2Code (FastAPI + JWT + hệ thống credit + Stripe thanh toán). Phần thay đổi chính là lõi AI — từ nhận diện biểu đồ sang nhận diện khuôn mặt và biển số xe qua camera IP. File này sẽ được cập nhật mỗi tuần khi hoàn thành các milestone.
